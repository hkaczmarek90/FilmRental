### Movie rental
Movie rental system

### Funcionalities
* Movie properties
    * Id
    * Title
    * Year
    * Rating

* Movie database
    * Check if movie is avaliable
    * Show all movies (id, all, avaliable, rating, year)

* User database
    * Show users
        * All
        * Active
        * Show rented movies

* Support rental
    * User can rent a movie
        * Only when movie is avaliable


* Park space
    * User can show history rentals
    * Money
    * Movie categories
    * Tag
